# Custom Module Example – Odoo

Este módulo fue desarrollado como ejercicio de aprendizaje técnico
para comprender la estructura interna de Odoo.

---

## Objetivo

- Entender la arquitectura de un módulo Odoo
- Explorar modelos, vistas y permisos
- Relacionar la parte técnica con el análisis funcional

---

## Alcance

- Modelo simple
- Vistas básicas
- Archivo de seguridad (`ir.model.access.csv`)

---

## Nota

Este módulo no forma parte del flujo funcional principal.
El foco del proyecto está puesto en el análisis funcional
y en la correcta implementación de procesos ERP.
